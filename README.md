# 📚 Repositório Nota 10 — Estudo para Concursos

## 🎯 Contexto e Objetivos

Este repositório foi criado com o objetivo de organizar e potencializar meus estudos para concursos públicos, utilizando ferramentas de Inteligência Artificial como apoio estratégico.

O foco principal é desenvolver um método eficiente de aprendizado baseado em:
- Revisão ativa
- Síntese de conteúdos
- Engenharia de prompts aplicada ao estudo

**Objetivos específicos:**
- Consolidar conteúdos recorrentes em concursos (ex: português, matemática, raciocínio lógico e atualidades)
- Criar um sistema de revisão inteligente com IA
- Desenvolver autonomia no uso de ferramentas como NotebookLM e ChatGPT
- Otimizar o tempo de estudo com técnicas modernas

## 🧠 Engenharia de Prompts e "Cicatrizes"

### 🔍 Prompts Utilizados

**Prompt 1:**
> Explique o conteúdo de forma simples como se fosse para um iniciante.

**Prompt 2:**
> Crie um resumo estruturado com tópicos e exemplos práticos.

**Prompt 3:**
> Gere 5 questões no estilo de concurso com gabarito comentado.

**Prompt 4:**
> Explique as pegadinhas mais comuns desse tema em provas.

---

### 🔄 Variações Testadas

- Ajuste de linguagem técnica → linguagem simples  
- Inclusão de exemplos práticos → melhor compreensão  
- Geração de questões → reforço do aprendizado ativo  

---

### ⚠️ Dificuldades Encontradas ("Cicatrizes")

- Respostas genéricas quando o prompt era pouco específico  
- Necessidade de definir nível (básico/intermediário/avançado)  
- Conteúdos fora do foco de concurso  

---

### 🛠️ Troubleshooting

| Problema                | Solução                                      |
|------------------------|---------------------------------------------|
| Resposta superficial   | Tornar o prompt mais específico             |
| Falta de exemplos      | Solicitar exemplos diretamente              |
| Conteúdo fora do edital| Incluir contexto do concurso no prompt      |
